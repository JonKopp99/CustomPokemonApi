A huge issue in this world is that not everyone can own a Pokemon.... With this API myself and others to create they're very own Pokemon!

## Structure

Every Pokemon gets certain attributes

 1. Name : String
 2. Type : String
 3. ImageURL : String
 4. Level : Int
